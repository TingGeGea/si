短信接口api条用注意事项！

发布时间：2016-05-15 14:39:13 | 发布者：shsixun.com | 查看:

思讯通短信接口注意事项

1.用户可访问 http://www.shsixun.com/ 自助注册账号、充值。

2.在思讯通上无须做任何配置即可直接发送短信，但通过在平台报备短信模板，审核通过后，发送速度将提高到5秒左右。故建议客户申请好模板后再进行测试。
3.群发短信将进入人工审核。故测试时尽量用单个号码，以保证接收速度。
短信格式要求
1.为防止骚扰，默认相同号码一天只能接收4条短信，如果测试期间使用短信较多，可以在后台白名单设置中添加，即可取消限制。
2.测试短信时，也请发送正规内容短信，如验证码、订单通知等，千万不要发送无意义的内容：例如 测一下、您好。
范例：【思讯通】您的验证码是1234,30秒内有效.若非本人操作请忽略此消息
3.短信内容中必须含短信签名格式为：【公司签名】 ，【】中间的文字可以是公司简称、网站简称、注册商标名称，3～10个字符，
如 好乐迪，志邦橱柜。不能是无意义的签名，例如【测一下】【试试看】等。

短信发送API

http://sms.shsixun.com/Port/default.ashx?method=SendSms&username=用户名&password =密码&phonelist=13658858&msg=
【思讯通】测试&SendDatetime=

USERNAME：在思讯通短信平台注册的用户名
PASSWORD：平台登录密码
PHONE：目标手机号码，多个手机号码用半角逗号分割
CONTENT：发送内容，采用UTF-8 URL ENCODE
返回 '0' 视为发送成功，其他内容为错误提示内容

短信接收API

http://xxx.xxx.xxx/xxx?m=PHONE&c=CONTENT
http://xxx.xxx.xxx/xxx：接收短信的URL地址
PHONE：发送方手机号码
CONTENT：短信内容，采用UTF-8 URL ENCODE
返回 '0' 视为接收成功，其他内容为错误提示内容

提示：须在思讯通管理后台 --> 设置 --> 短信设置中，配置 “短信上行接收地址” ，思讯通才会开始推送回复短信到指定网址
